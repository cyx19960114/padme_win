Guidelines for Train Image Approval Process:

The Train image approval will initiate a merge request to the main branch in the Official Train Store repository.

The approval is only possible if all of these conditions are met:

  • You have one of the following access level in the project: Owner and Maintainer.
  • The selected Train image has received an average rating of 4.0 through ratings and feedback.
  • The selected GitLab branch is any other branch than the main branch.

Click on Give Feedback to record a rating and a comment for the selected Train image.
