Welcome to PADME!

Thank you for your registration. Your station is in the on-boarding queue in our central service now. In order to finish the on-boarding process, you have to follow the step-by-step manual below. If any questions arise, feel free to contact us via: pht@dbis.rwth-aachen.de

1

Install Docker on your local machine.

Get Docker!
2

Download the installation files and extract them. Navigate to the folder. Open a console in this directory and type:

docker-compose up -d

This will start the Station Software!

Enter the Station Software!
3

The Station Software includes a configuration guide, which will show up if the Station Software is booted the first time. This configuration guide will finalise the secure on-boarding process.

For the configuration guide, you need the encrypted file including your personal credentials. Please download this encrypted file.

The configuration guide will guide you through the last steps!

For further information, we refer to our website and our official web portal. We provide news and also release notes. Stay tuned!
