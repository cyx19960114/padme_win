Guidelines for providing Additional Metadata:

Use this form to add additional metadata to the Train image as a key-value pair. For example:

  • train_creator : name_of_the_creator
  • published_date : current_date_time
  • analysis_algorithm : algorithm_name

Click on Add New Row to add more metadata as key-value pair.
